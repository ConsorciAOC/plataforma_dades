
# 📘 Plataforma de Dades

Aquest repositori recopila diversos **exemples, guies i instructius** per treballar amb la **plataforma de dades implementada a Microsoft Fabric**, incloent la connexió al SQL Endpoint, integració amb Power BI i exemples de programació en Python per a l'accés autenticat.


## 📦 Materials de suport
Cada carpeta inclou totes les imatges necessàries per seguir els passos descrits als instructius.

---
## 📌 Objectiu del repositori
Aquest repositori té com a finalitat:
- Proporcionar **exemples pràctics** per equips tècnics.  
- Servir com a **base de coneixement** per projectes a Microsoft Fabric.  
- Unificar bones pràctiques de connexió, consum i gestió de dades a la plataforma.
